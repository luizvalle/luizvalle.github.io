RLSA Library

A Python library available for installation on the Python Package Index (PyPI)

I made this library when I could not find any other that suited the needs of a project I was working on to extract paragraphs from a scanned document. It is an implementation of the Run Length Smearing Algorithm used to merge adjacent words/lines in an image.

The user simply specifies the horizontal and vertical distance thresholds that determine which pixels get merged. This implementation works with different page layouts (single column vs mutiple columns).
